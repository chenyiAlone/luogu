## 问题总结

1. 数据范围开数组 **一定要够大** ，c++ 没有越界提醒，不然最明显的效果是超时

2. **`for`** 循环
    -  **边界** 和 **更新** 一定要一直，写反了就死循环了
    - 循环变量一定不要用错了

3. 函数中的坑
    - `<cstring>: memcpy(des, src, sizeof(src))` 这里必须是 **`src`** 的全长
